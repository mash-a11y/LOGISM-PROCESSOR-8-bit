# **8-Bit CPU Design and Simulation**
# **Overview**
This project presents the design and simulation of an 8-bit CPU developed using Logisim-evolution. The objective of the project is to demonstrate the fundamental working principles of computer architecture, digital logic design, and processor execution cycles in an educational environment.The CPU was designed by integrating essential hardware components such as registers, arithmetic logic units (ALU), control units, memory interfaces, and clock-driven operations. The project helps students understand how instructions are processed at the hardware level and how data flows through different processor components.
This implementation serves as a learning-oriented simulation of a simple processor architecture for academic and educational purposes.
# **Features**
- 8-bit processor architecture
- Arithmetic Logic Unit (ALU)
-Register-based data storage
- Clock-driven instruction execution
- Control unit for instruction management
- Basic instruction handling
- Memory and data flow simulation
- Digital logic circuit implementation
- CPU simulation using Logisim-evolution
# **Components Used**
- Registers
- Multiplexers
- Adders and Subtractors
- Logic Gates
- Program Counter
- Instruction Decoder
- Arithmetic Logic Unit (ALU)
- Clock Components
# **Technologies Used**
- Logisim-evolution
- Digital Logic Design
- Computer Architecture Concepts
